HOW TO PLAY:

Use ⬆️ ⬇️ ⬅️ ➡️ to control Pac-man

Keep the key down for smooth movement

Press "space" for pausing the game and to return back to the game

Ghosts are wandering around a while and then start chasing Pac-man

If Pac-man eats a power-pellet, ghosts become "scared" for 10 seconds

Scared ghosts will blink the last 3 seconds before returning to normal

Pac-man has to clear the whole gameboard from dots and pellets to win the game

If you run out of lives, game over


SCORING:

1 point for pac-dot

10 points for power-pellet

Doubling points for scared ghosts eaten within 10 seconds:

  • 200 points for the first scared ghost eaten
  • 400 points for the second
  • 800 points for the third
  • 1600 points for the fourth

NOTE: The points for scared ghosts reset to 200 when a power-pellet is eaten.

PAUSED

Press Space to Resume

Press R to Restart
